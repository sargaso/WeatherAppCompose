# WeatherApplicationCompose
Разработка приложения для отображения погоды на Jetpack Compose. Для получения данных погоды использовал открытое API.

## Технологический стек
* MVVM
* Jetpack Compose
* Volley
* Pager layouts - Accompanist 
* Google Maps SDK
* Viewbinding
